Caracteristicas del servidor apache

Apache es un servidor web con soporte para HTTP 1.1 y además, logra añadir soporte para HTTP 2 según la norma RFC 7540. Entre las principales características de Apache, se encuentran las siguientes:

  1. Es gratuito y de fuente abierta
  2. Instalación y configuración sencilla
  3. Altamente extensible y adaptable mediante módulos
  4. Funciones incorporadas para autenticación y validación de usuarios
  5. Soporte para los diferentes tipos de lenguajes de programación como Perl, PHP y Python

Si aún no conoces mucho sobre estos lenguajes que se desarrollan, te sugerimos inscribirte en cursos de fundamentos de JavaScript u otros lenguajes como Python, R, etc., con el objetivo de que tengas una base más sólida en este universo de la programación y lo digital


Versiones del servidor apache

Apache HTTP Server ha tenido varias versiones a lo largo de su historia. A continuación, te proporciono una lista de algunas de las versiones más destacadas hasta la fecha de mi conocimiento en septiembre de 2021:

  1. Apache 0.9.6 (1995): Esta fue la primera versión de Apache, pero no se considera la primera versión estable. Marcó el inicio del proyecto.
  2. Apache 1.0 (1996): Esta fue la primera versión estable de Apache y marcó el comienzo de su popularidad en la comunidad de servidores web
  3. Apache 1.3 (1998): Esta fue una de las versiones más populares y utilizadas de Apache durante muchos años. Introdujo una serie de mejoras en comparación con la versión 1.0.
  4. Apache 2.0 (2002): Esta versión trajo importantes cambios en la arquitectura y rendimiento, incluyendo el soporte para múltiples hilos y procesos, así como una estructura modular.
  5. Apache 2.2 (2005): Continuando con las mejoras de la versión 2.0, la 2.2 mejoró la seguridad y agregó características como la autorización basada en la dirección IP y una mayor flexibilidad en la configuración.



Requerimientos del servidor apache

Los requisitos para instalar y ejecutar Apache HTTP Server pueden variar según la plataforma y el sistema operativo en el que desees utilizarlo. En general, los requisitos básicos incluyen:

  1. Un sistema operativo compatible, como Linux, Windows, macOS, o BSD
  2. Acceso a Internet o una red local si deseas que el servidor sea accesible desde la web.
  3. Hardware adecuado, que puede variar según la carga de tráfico prevista. En general, una computadora con suficiente memoria y CPU es suficiente para fines de desarrollo o sitios web de bajo tráfico
  4. - Software adicional, como PHP, MySQL (o su alternativa, como MariaDB) si deseas habilitar funcionalidades específicas en tu servidor web.
